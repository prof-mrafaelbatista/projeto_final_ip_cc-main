Fundamentos da Progamação

Estruturas de Seleção

As estruturas de seleção servem para tomar decisões no código, ou seja, executar certos blocos de código dependendo de uma condição. Em Python, usamos principalmente:

  • If
  • Elif
  • Else

Exemplo:
idade = 18
if idade >= 18:
print("Você é maior de idade.")
else:
print("Você é menor de idade.")

Estruturas de Repetição

As estruturas de repetição permitem executar um bloco de código várias vezes, com base em uma condição ou sequência de valores. Principais tipos em Python:

  • For(laço com contador ou para percorrer listas)
  • While (laço com condição)

Exemplo For:
for i in range(5): # Vai ser mostrado na tela os números de 0 a 5
print(i)

Exemplo While:
contador = 0
while contador < 5:
print(contador)
contador += 1

Vetores e matrizes

  • FVetores (ou listas) são estruturas que armazenam vários elementos em uma única variável.
  • Matrizes (ou listas de listas) são listas dentro de listas, como uma tabela (linhas e colunas).

Exemplo Vetor:
frutas = ["maçã", "banana", "laranja"]
print(frutas[0]) # imprime "maçã"

Exemplo Matriz
matriz = [
[1, 2, 3],
[4, 5, 6]
] print(matriz[1][2])

Funções e procedimentos

Funções são blocos de código reutilizáveis que recebem entradas (parâmetros), realizam algum processamento e retornam um resultado.
Procedimentos são funções que não retornam valores, apenas executam ações (como exibir algo).

Exemplo Função:
def somar(a, b):
return a + b
resultado = somar(3, 4)
print(resultado)

Exemplo Procedimento: def exibir_mensagem():
print("Olá, seja bem-vindo!")
exibir_mensagem()

Tratamento de exceções em Python

Exceções são erros que ocorrem durante a execução de um programa. Se não forem tratadas, elas fazem o programa parar com uma mensagem de erro.

  • Try -> Tenta executar um bloco de código
  • Except -> Executa um bloco alternativo se um erro ocorrer
  • Else -> Executa se nenhuma exceção ocorrer no bloco try
  • Finally -> Executa sempre, com ou sem erro

Exemplo Try e Except:
try:
numero = int(input("Digite um número: "))
resultado = 10 / numero
print("Resultado:", resultado)
except ZeroDivisionError:
print("Erro: divisão por zero não é permitida.")
except ValueError:
print("Erro: você não digitou um número válido.")

Exemplo Else e Finally
try:
n = int(input("Digite um número: "))
except ValueError:
print("Isso não é um número.")
else:
print("Você digitou o número:", n)
finally:
print("Fim do programa.")
